# Mon Portfolio - Kylian

Bienvenue sur le dépôt de mon portfolio. Ce projet est structuré pour montrer mon évolution technique à travers différentes étapes de ma formation.

## Structure du Projet (Branches)

Le projet est divisé en deux versions majeures accessibles via les branches Git :

### [portfolio-v1](https://github.com/BZHKylian/Portfolio/tree/portfolio-v1) (BUT 1ère année)
* **Objectif :** Première version réalisée lors de ma première année de BUT Informatique.
* **Stack :** `HTML5`, `CSS3`, `JavaScript` (Vanilla).
* **Focus :** Apprentissage des bases du web, responsive design statique.
* **URL :** [https://portfolio-v1.kylianhouedec.fr](https://portfolio-v1.kylianhouedec.fr)

### [portfolio-v2](https://github.com/BZHKylian/Portfolio/tree/portfolio-v2) (Version Actuelle)
* **Objectif :** Refonte globale visant à moderniser l'interface et optimiser l'architecture pour rendre l'ajout de projets quasi instantané.

* **Stack :** `Next.js`, `Tailwind CSS`.
* **Focus :**
    - *Architecture de données* : Utilisation de fichiers de constantes JS pour piloter l'affichage des projets dynamiquement.

    - *Composants réutilisables* : Développement de composants UI atomiques avec Tailwind.

    - *Performances* : Optimisation SEO pour une visibilité maximale.

* **URL :** [https://kylianhouedec.fr](https://kylianhouedec.fr)
